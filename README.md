# Mcxiaocaibug.github.io

Mcxiaocaibug 的个人网站：写代码，也写风与月。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

项目使用 Next.js 静态导出，推送到 `main` 分支后由 GitHub Actions 发布到 GitHub Pages。

页面字体为 [Maple Mono](https://github.com/subframe7536/maple-font)，并基于实际使用字符生成了网页子集；字体许可见 `app/fonts/LICENSE.txt`。
